# To-Do List App

A simple, interactive To-Do List web application built with Python and Flask. Users can add tasks, mark them as completed, and delete tasks.

---

## Features

* Add tasks with a text input field.
* Mark tasks as completed with a checkbox.
* Delete tasks from the list.
* Rain animation with wind effect - taken from https://codepen.io/arickle/pen/XKjMZY with slight tweaks i made myself
* Responsive design and user-friendly interface.

---

## Technologies Used

* **Frontend:** HTML, CSS, JavaScript, jQuery
* **Backend:** Python, Flask
* **Styling:** CSS3 animations for rain effects

---

## Installation & Setup

1. **Clone the repository:**

```bash
git clone https://github.com/USERNAME/REPO_NAME.git
cd REPO_NAME
```

2. **Create a virtual environment:**

```bash
python -m venv venv
```
3. **Install dependencies:**

```bash
pip install -r requirements.txt
```

5. **Run the application:**

```bash
flask run
```
